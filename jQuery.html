JavaScript Libraries and Frameworks

Many developers who work with JavaScript rarely use JavaScript alone. Rather, they use libraries and frameworks that abstract out much of the complexity of the underlying language.

JQuery

jQuery is a popular, free, open-source JavaScript library that makes navigating and manipulating the DOM easier than just using JavaScript.

Angular

Angular, which was developed at Google. The current version is a different framework from AngularJS, which it effectively replaces. They are quite different, AngularJS was based on JavaScript, whereas Angular is built on TypeScript.

Bootstrap

Bootstrap was first developed at Twitter and focuses on page presentation, particularly in a mobile context. It is really a CSS library rather than a JavaScript one, but does have features that use JavaScript.
